# Exgravatar

An Elixir module for generating [Gravatar](http://gravatar.com) urls.

## Usage

```elixir
Exgravatar.gravatar "jdoe@example.com"
#=> "http://gravatar.com/avatar/694ea0904ceaf766c6738166ed89bafb"
```
